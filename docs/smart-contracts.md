# Across Smart Contracts

Across smart contracts including the `HubPool.sol` and `SpokePool.sol` comprising the Hub and Spoke architecture of the Across protocol are all found in https://github.com/across-protocol/contracts

All smart contracts that are designed to be used by users of the Across network can be found in across-protocol/contracts.

Any smart contracts found in this repo in /contracts/ are designed to be used by the bots in this repo.
